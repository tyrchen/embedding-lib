# Idea

I want to create an easy to use crate to do text embeddings, leveraging ./text-embeddings-inference code. The API should look like this:

```rust
struct TextEmbeddings {
  infer: Infer, // use Infer from text-embeddings-inference core crate
}

struct TextEmbeddingsOptions {
  ...
}

impl TextEmbeddings {
  pub fn new(options: TextEmbeddingsOptions) -> Self; // should extract core code from text_embeddings_router::run()

  pub fn embed(&self, texts: &[&str]) -> Result<Vec<Vec<f32>>, Error>;
}
```
